# -skill
【一句话介绍】 把一堆杂乱无章的发票PDF丢给AI，即可自动规范发票命名+输出Excel 报表，重复和异常发票一目了然，提升报销效率  【核心能力】 1、智能提取：自动从发票中识别发票号码、金额、日期、抬头税号等关键字段 2、重复检测：自动找出重复发票，文件名加“重复”前缀，避免重复报销 3、自动分类：按消费类型归类（餐饮、交通、办公用品、酒类等） 4、抬头校验：按需触发，提供公司名称和税号即可启用，支持多抬头校验场景 5、批量重命名：统一为 金额-日期.pdf 格式，重命名前预览，确认后才执行 6、Excel报表：生成多个sheet，包含发票明细、发票类别、购买方记录、重复发票等，数据一目了然
