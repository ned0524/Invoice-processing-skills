【一句话介绍】
把一堆杂乱无章的发票PDF丢给AI，即可自动规范发票命名+输出Excel 报表，重复和异常发票一目了然，提升报销效率

【核心能力】
1、智能提取：自动从发票中识别发票号码、金额、日期、抬头税号等关键字段
2、重复检测：自动找出重复发票，文件名加“重复”前缀，避免重复报销
3、自动分类：按消费类型归类（餐饮、交通、办公用品、酒类等）
4、抬头校验：按需触发，提供公司名称和税号即可启用，支持多抬头校验场景
5、批量重命名：统一为 金额-日期.pdf 格式，重命名前预览，确认后才执行
6、Excel报表：生成多个sheet，包含发票明细、发票类别、购买方记录、重复发票等，数据一目了然

【操作指引】
1、在微信workbuddy（或者电脑端workbuddy）发出指令：“桌面上有几个发票，给我放到一个统一的文件夹，并且整理一下”
2、workbuddy的claw就会自动运行，最终输出的是一份excel报表以及将所有发票按照规定的格式命名
3、处理6个发票的时间大概是2min

【安装方式】
1、直接问workbuddy，让它给我们在电脑上打开workbuddy/skills的目录
2、下载该skill的压缩包，把解压后的文件夹拖到上面的目录下
3、重启一下workbuddy，关掉当前对话，开一个新对话。AI 启动时会自动扫描 skills 目录，识别到新的 skill
4、不需要手动加载或配置，只要对话里提到发票相关的内容（比如"帮我整理发票"、"处理发票"），AI 就会自动触发这个 skill
